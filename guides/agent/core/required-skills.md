# Required Skills

- Any agent performing code review must load and use `code-reviewer`.
- React Web Developer must always use `vercel-react-best-practices`.
- React Web Developer must use `frontend-design` for UI work.
- React Web Developer must use `web-design-guidelines` for UI/UX/accessibility review.
- React Web Developer must use `tailwind-design-system` whenever Tailwind is used.
- Backend API Specialist must load and use `hono`.

See [../agent-skills-and-docs.md](../agent-skills-and-docs.md) for skill usage and doc hygiene details.
