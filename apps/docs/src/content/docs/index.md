Time Tracker is workforce time and attendance software with two deployment models:

- Hosted by the vendor
- On-premise deployment in your own environment

The product has:

- A web application for managers and employees
- A REST API documented with an OpenAPI specification

## What to read first

- Follow [Deployment Quickstart](/quickstart) to choose hosted or on-premise setup.
- Review [Web App Basics](/web-app/basics) for day-to-day usage.
- Use [REST API and OpenAPI](/api/rest-openapi) to integrate external systems.
